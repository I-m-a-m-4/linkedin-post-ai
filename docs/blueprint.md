# **App Name**: LinkedIn Post Formatter AI

## Core Features:

- AI-Powered Formatting: Automatically formats pasted text with line breaks, bolding, emojis, bullet points, and headline generation using generative AI. It decides when to add line breaks, bold certain texts and what kind of expressions suit the post, therefore making use of AI tools.
- Live LinkedIn Preview: Displays a real-time preview of how the formatted text will appear on LinkedIn (desktop and mobile).
- Manual Formatting Controls: Provides basic manual controls for bold, italics, underline, bullet points, numbering, spacing, and emoji insertion.
- AI Formatting Refinement: Offers a button to 'Shuffle formatting ideas' that applies alternative AI formatting styles (bolder, minimal, conversational) using generative AI tool to decide what suits the formatting refinement.

## Style Guidelines:

- Primary color: LinkedIn blue (#0077B5) to maintain brand recognition.
- Background color: Light gray (#F2F2F2) for a clean, professional look.
- Accent color: A brighter blue (#00A0DC) for buttons and interactive elements.
- Body and headline font: 'Inter', a grotesque-style sans-serif with a modern, machined, objective, neutral look suitable for headlines or body text.
- Simple, professional icons for formatting controls (bold, italic, bullet points, etc.).
- Clean and intuitive layout with the input text area on one side and the live preview on the other.
- Subtle transitions when applying AI formatting or switching between preview modes.
